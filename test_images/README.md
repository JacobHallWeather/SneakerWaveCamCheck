# Test Images

This folder is for uploading beach images for testing purposes. Below are instructions on how to add images to this folder using different methods:

## Uploading via GitHub Web Interface
1. Navigate to the `test_images` folder in your repository.
2. Click on the `Add file` button and select `Upload files`.
3. Drag and drop your beach images or choose the files from your computer.
4. Scroll down to commit the changes and add an optional commit message.
5. Click on `Commit changes` to finalize the upload.

## Uploading using Codespaces
1. Open your repository in GitHub Codespaces.
2. In the terminal, navigate to the `test_images` folder if it exists, or create it using `mkdir test_images`.
3. Add your beach images to the folder by dragging and dropping them or using command line tools.
4. Use `git add .` to stage the new images.
5. Commit your changes with `git commit -m "Add beach images for testing"`.
6. Push your changes to the repository using `git push`.

## Uploading via Git Command Line
1. Clone your repository if you haven't already: `git clone https://github.com/JacobHallWeather/SneakerWaveCamCheck.git`.
2. Navigate to the `test_images` folder (`cd SneakerWaveCamCheck/test_images`). If it doesnâ€™t exist, create it with `mkdir test_images`.
3. Copy or move your beach images into the `test_images` folder.
4. Use `git add test_images/*` to stage the images for commit.
5. Commit your changes using `git commit -m "Add beach images for testing"`.
6. Push your changes back to GitHub with `git push`.
